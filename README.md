# 🎙️ VoiceArena

> AI-powered Sports Tournament Management Platform

## 📖 Overview

VoiceArena is a modern SaaS platform designed to manage sports tournaments, teams, players, and matches. It also aims to provide AI-powered voice commentary, analytics, and reporting.

---

## 🚀 Current Features

- 🔐 JWT Authentication
- 👤 User Management
- 🏢 Organization Management
- 🛡️ Role-Based Access Control (RBAC)
- ⚡ FastAPI REST APIs
- 🗄️ PostgreSQL Database
